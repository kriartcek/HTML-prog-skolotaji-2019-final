HTML elementi

Šodien mēs iemācījāmies kas ir HTML elementi un kā ar tiem darboties.

Mēs jau esam paspējuši apskatīt tādus HTML elementus, kā:
  • Konteiners "< div >"
  • Virsraksts "< h1 >, < h2 >, ..., < h6 >"
  • Rindkopa "< p >"
  • Atribūts "< a >"
  • Rindas atdalītājs "< br >"
Tagad apkskatīsim vēl šos elementus:
  1. Nenumurēts saraksts "< ul >"
  2. Saraksta objekts "< li >"
  3. Numurēts saraksts "< ol >"
  4. Attēls "< img >"
  5. Poga "< button >"
  6. Treknraksts "< b >"

Vairāk informācijas par visiem HTML elementiem mēs varam atrast w3schools mājaslapā.
